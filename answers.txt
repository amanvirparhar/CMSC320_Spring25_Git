Q1: Why might we consider using Git?

Answer: Version control software, like Git, allow us to easily keep track of the modifications we make to our codebase. Git is also really widespread in its usage, which is why platforms like GitHub and GitLab have sprung up; these platforms help developers from around the world share and modify each others' code.

Q2: What field could data science be applied to? Give an Example.

Answer: Data science has many applications in the healthcare field. This is because patterns that are apparent in data, as well as predictions that can be made from data, are key in helping researchers understand how different diseases and disorders work. This knowledge can help these researchers develop cures to these conditions, which include various types of cancers, autoimmune disorders, and more.

Q3: What is the difference between Merging and Rebasing?

Answer: When you merge two branches, all of the commits in both of the branches are kept. However, when you rebase two branches, the history of the branch you're on is modified to include all of the commits of the previous branch along with the commits that modify that branch.
